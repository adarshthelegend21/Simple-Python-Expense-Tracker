Project Title
Simple Python Expense Tracker

Overview of the Project
A basic command-line application to track personal expenses. Users can add expense entries with date, amount, category, and description. It also displays all recorded expenses and computes the total amount spent.

Features

Add expense with details (date, amount, category, description)

Display all expenses in a list

Show the total sum of all expenses

Simple menu-driven interface

Easy to use and extend

Technologies/Tools Used

Python 3 (standard library)

No external dependencies

Steps to Install & Run the Project

Ensure Python 3 is installed on your system.

Download or clone this repository to your local machine.

Open a terminal/command prompt in the project directory.

Run the program with the command: python filename.py (replace filename.py with the actual script name).

Instructions for Testing

Run the program and choose option 1 to add various expenses.

Use option 2 to verify all expenses are correctly listed.

Use option 3 to check the total expenses sum.

Select option 4 to exit the program.

Try entering invalid options to test error handling.
