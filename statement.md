Problem Statement
Managing personal expenses manually can be tedious and error-prone. There is a need for a simple tool that helps users record, view, and calculate their spending easily to improve budgeting and financial awareness.

Scope of the Project
This project provides a basic command-line expense tracker allowing users to input expense details including date, amount, category, and description. It supports listing all expenses and calculating the total spending. The project does not handle user accounts or persistent storage but serves as a foundational learning tool.

Target Users
Students and beginners learning Python programming

Individuals seeking a simple expense tracking method without complex features

Users interested in learning basic input/output and data handling in Python

High-level Features
Add expense entries with date, amount, category, and description

Display a list of all recorded expenses

Compute and show the total amount of all expenses

Simple menu-based interface for easy navigation
